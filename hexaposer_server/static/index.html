• GainNode

    Mit der GainNode-Schnittstelle können Lautstärkeänderungen vorgenommen werden.

  • DelayNode

    Mit der DelayNode-Schnittstelle kann eine Verzögerung hinzugefügt werden.

  • PannerNode

    Die PannerNode-Schnittstelle entscheidet aus welcher Richtung die Soundausgabe kommt.

  • BiquadFilterNode

    Mit der BiquadFilterNode-Schnittstelle(Typ:Lowpass) Signale mit Frequenzen unterhalb ihrer Grenzfrequenz ungeschwächt passieren gelassen, während Anteile mit höherer Frequenzen gedämpft werden.

  • WaveshaperNode

    Mit dem WaveshaperNode-Modul werden Audiosignale mit einer dynamischen Verzerrung hinterlegt.
